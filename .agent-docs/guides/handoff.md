# Handoff Guide

Handoff is a compact snapshot of the current state designed to survive chat
resets and long runs.

## File
- `.agent-docs/memory/HANDOFF.md`

## When to Update
- After major milestones.
- Before switching scope.
- Before ending a long session.

## Rules
- Keep it short and high-signal.
- Link to `PLANS.md`, auto-context, ADRs, and key files.
- Include the last verification commands and outcomes.
- Include a "Next Chat Prompt" for continuity.
