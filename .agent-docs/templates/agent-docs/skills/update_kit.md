# Update Kit

Use this guide to keep the automation kit aligned with upstream.
1. Read `KIT_VERSION.md` and capture the current version.
2. Compare to upstream releases or commits.
3. Apply `non_destructive_bootstrap` using the merge protocol.
4. Run `skills_auditor` and `skills_sync`.
5. Refresh indexes and log actions.
6. Update `KIT_VERSION.md` with the new version data.
7. Use the update kit checklist for completeness.
