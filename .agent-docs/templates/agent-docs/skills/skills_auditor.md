# Skills Auditor

Use this skill to detect incorrectly installed skills.
1. Scan `.codex/skills/*/SKILL.md` for missing frontmatter or mismatched names.
2. Compare `.codex/skills/` and `.agent-docs/skills/` for gaps.
3. Record findings in `.agent-docs/memory/SKILLS_STATUS.md`.
4. Propose fixes using merge protocol.
