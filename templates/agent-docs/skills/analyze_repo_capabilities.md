# Analyze Repo Capabilities

Use this skill to detect existing tooling and activate compatible workflows.
1. Scan for signatures (CI, Docker, IaC, frameworks, agents).
2. Record findings in `.agent-docs/memory/CAPABILITIES.md`.
3. Update `COMMANDS.md`, `COMMANDS.json`, and `MANIFEST.yaml`.
4. Update `INDEX.md` and `INDEX.json`.
5. Activate relevant skills instead of overwriting conventions.
6. Note gaps or conflicts for follow-up.
